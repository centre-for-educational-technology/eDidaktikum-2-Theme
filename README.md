## To get started:

1. Install the dependencies:

   ```bash
   # Using npm
   npm install
   ```

2. Start the development server:

   ```bash
   # Using npm
   npm run serve
   ```

This will generate a `build` directory with the compiled CSS file.

Now you should be able to see the project running at localhost:8080
